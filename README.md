# IELTS Vocabulary Web App v2.8 All-in-one

这是 v2.8 综合升级版，包含 v2.2 - v2.8 的主要学习体验升级。

## 新增 / 优化

### v2.2 今日目标 + 本组完成页
- 首页设置今日目标；
- 本地记录今日完成量；
- 每组学习完成后显示反馈：完成数、正确率、忘记、模糊、熟练。

### v2.3 复习算法升级
- 复习等级从 0-8；
- 忘记、模糊、正确、熟练对应不同间隔；
- 返回下一次复习间隔提示。

### v2.4 薄弱词专项训练
- 新增“薄弱”入口；
- 专练错过、模糊、低等级、困难标记词。

### v2.5 章节进度
- 新增章节进度页；
- 显示每章总词数、已学、掌握、到期、完成百分比。

### v2.6 PWA 手机端优化
- 增加 manifest；
- 增加图标；
- 注册 service worker；
- 可添加到手机主屏幕。

### v2.7 Web 管理概览
- 新增 Web 管理概览；
- 用 ADMIN_PIN 查看用户数、词数、学习记录数、故事数和章节词数；
- 公共词库导入/重置仍建议继续使用 Streamlit 后台，避免误操作。

### v2.8 AI式故事增强
- 故事从单句模板升级为科研场景记忆路线；
- 中英双语；
- 附快速自测。

## 部署

覆盖 GitHub 仓库后 Push，Vercel 会自动部署。

## 环境变量

保持原来的 Vercel 环境变量即可：

```env
DATABASE_URL=...
AUTH_SECRET=...
PIN_PEPPER=ielts-vocabulary-app-v1.2
ADMIN_PIN=...
NODE_ENV=production
```
