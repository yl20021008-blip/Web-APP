# IELTS Vocabulary Web App v2.9 One-click Learning Flow

这是 v2.9 一键学习流升级版。

## 新增功能

### 1. 一键开始今天的学习
首页新增“开始今天的学习”，不需要用户判断该学新词、复习还是练薄弱。

### 2. 智能混合队列
系统自动混合：

```text
到期复习
薄弱词
今日新词
```

优先级约为：

```text
复习 > 薄弱 > 新词
```

### 3. 错词本组回炉
如果用户点击“完全忘记”，该词会被重新插入到本组后面，再次出现。

### 4. 每日总结
新增“总结”页面：

```text
今日完成
正确率
忘记数
模糊数
明日复习
建议重点回看词
```

### 5. 更省脑子的学习路径
用户只需要：

```text
打开 APP
点击 一键
一路判断记忆状态
看答案
下一词
最后看总结
```

## 部署

覆盖 GitHub 仓库后 Push，Vercel 自动重新部署。
