# IELTS Vocabulary Web App v2.1 Recall-first SRS

这是 v2.1 记忆逻辑升级版。

## 本版本修复的核心问题

之前学习页会先显示中文释义、例句和读音，然后用户再点“忘记 / 模糊 / 正确 / 熟练”。

这会导致：

```text
用户没有真正回忆
评分不准确
复习算法失真
```

v2.1 改成标准 SRS 逻辑：

```text
Step 1：只显示英文单词
Step 2：用户先回忆
Step 3：用户选择 完全忘记 / 有点模糊 / 基本记得 / 非常熟练
Step 4：系统记录结果
Step 5：显示答案、音标、读音、例句、翻译
Step 6：点击下一词
```

## 部署方式

1. 解压本 ZIP；
2. 覆盖 GitHub 仓库根目录；
3. Commit；
4. Push；
5. Vercel 会自动重新部署。

## 环境变量保持不变

不用改 Vercel 环境变量。
