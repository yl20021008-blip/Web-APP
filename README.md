# IELTS Vocabulary Web App v3.0 雅思四技能词汇训练版

这是 v3.0 升级版，目标是把 APP 从“背单词工具”升级为“雅思词汇训练器”。

## 核心升级

原来的背词目标主要是：

```text
英文 → 中文
```

v3.0 新增雅思训练入口，覆盖：

```text
看得出
想得起
听得懂
拼得对
写得出
说得出
```

## 新增训练模式

### 1. 阅读识别 Reading Recognition

```text
英文 → 中文
```

适合阅读中快速认词。

### 2. 写作回忆 Writing Recall

```text
中文释义 → 英文
```

适合训练写作和口语中的主动调用。

### 3. 听音辨词 Listening Recognition

```text
只播放读音 → 猜单词
```

适合雅思听力反应速度。

### 4. 拼写检查 Spelling

```text
中文 / 读音 → 输入英文拼写
```

适合减少听力填空和写作拼写错误。

### 5. 例句填空 Sentence Cloze

```text
例句挖空 → 回忆目标词
```

适合训练语境、搭配和写作用法。

### 6. 口语30秒 Speaking Output

```text
3–5个目标词 → 30秒口语回答
```

适合把词汇转化成口语输出能力。

## 推荐每天使用流程

```text
20分钟：一键学习
15分钟：写作回忆
15分钟：听音辨词
10分钟：拼写检查
15分钟：例句填空
10分钟：口语30秒
```
