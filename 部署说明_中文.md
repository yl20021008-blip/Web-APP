# v3.0 雅思四技能词汇训练版部署说明

## 操作步骤

1. 解压 ZIP；
2. 打开 GitHub Desktop；
3. Repository → Show in Explorer；
4. 把 v3.0 文件夹里的全部内容复制进去覆盖；
5. Summary 填：

```text
Upgrade to v3.0 IELTS four skills training
```

6. Commit to main；
7. Push origin；
8. Vercel 自动重新部署。

## 测试重点

部署后重点测试底部导航里的：

```text
雅思
```

里面包含 6 个训练模式：

```text
阅读识别
写作回忆
听音辨词
拼写检查
例句填空
口语30秒
```

环境变量不用改。
