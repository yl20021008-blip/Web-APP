# v2.0 Web App 部署说明

## 你要做的事

### 第一步：新建 GitHub 仓库

建议新建一个仓库，不要和 Streamlit 混在一起。

仓库名可以叫：

```text
ielts-vocab-webapp
```

### 第二步：上传文件

用 GitHub Desktop：

```text
File → Add Local Repository
或者 Create New Repository
```

然后把这个压缩包解压后的全部文件复制进去。

提交：

```text
Commit summary:
Deploy Next.js web app v2
```

然后：

```text
Push origin
```

### 第三步：Vercel 部署

进入 Vercel：

```text
Add New Project
↓
Import Git Repository
↓
选择你的 GitHub 仓库
↓
Framework Preset: Next.js
```

不用填 `main_app/app.py`。

### 第四步：填环境变量

在 Vercel Project 的 Environment Variables 里填：

```env
DATABASE_URL = 你的 Supabase 数据库 URL
AUTH_SECRET = 你自己设置的一串长密码
PIN_PEPPER = ielts-vocabulary-app-v1.2
ADMIN_PIN = 你的管理员PIN
NODE_ENV = production
```

`DATABASE_URL` 可以复制你 Streamlit 里已经能用的那条。

### 第五步：Deploy

点 Deploy。

部署成功后，Vercel 会给你一个网址，例如：

```text
https://ielts-vocab-webapp.vercel.app
```

以后普通用户用这个网址学习。

### 第六步：继续保留 Streamlit 后台

你的 Streamlit v1.5.1 继续用于：

```text
公共词库管理
预设词库
词汇补全
管理员操作
```

Web App 负责：

```text
普通用户学习
复习
故事
统计
我的词库
```

这是目前最现实、最稳定的结构。
