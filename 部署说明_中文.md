# v2.1 Recall-first SRS 部署说明

## 操作步骤

1. 解压 ZIP；
2. 打开 GitHub Desktop；
3. Repository → Show in Explorer；
4. 把 v2.1 文件夹里的所有内容复制进去覆盖；
5. Summary 填：

```text
Upgrade study flow to recall-first SRS
```

6. Commit to main；
7. Push origin；
8. Vercel 会自动重新部署。

## 升级后学习页逻辑

先显示：

```text
单词
完全忘记 / 有点模糊 / 基本记得 / 非常熟练
```

点击任意按钮后：

```text
记录学习结果
显示中文释义
显示音标和读音
显示例句和翻译
出现“下一词”
```

环境变量不用改。
