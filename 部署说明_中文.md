# v2.0.1 SSL 修复版部署说明

## 你的报错

```text
self-signed certificate in certificate chain
```

这是 Web App 连接 Supabase PostgreSQL 时的 SSL 证书校验问题。

## 解决方式

上传 v2.0.1 修复版代码到 GitHub，Vercel 会自动重新部署。

## 操作步骤

1. 解压 ZIP；
2. 打开 GitHub Desktop；
3. Repository → Show in Explorer；
4. 把 ZIP 里的所有内容复制进去覆盖；
5. Summary 填：

```text
Fix Supabase SSL connection
```

6. Commit to main；
7. Push origin；
8. 回到 Vercel 等它自动 Redeploy。

## 环境变量不用改

继续用你刚才填的：

```text
DATABASE_URL
AUTH_SECRET
PIN_PEPPER
ADMIN_PIN
NODE_ENV
```

## 如果仍然失败

去 Vercel：

```text
Project → Deployments → 最新部署 → Logs
```

把最底部红色错误截图发给我。
