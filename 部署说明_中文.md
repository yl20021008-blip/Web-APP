# v2.8 综合升级版部署说明

## 操作步骤

1. 解压 ZIP；
2. 打开 GitHub Desktop；
3. Repository → Show in Explorer；
4. 把 v2.8 文件夹里的所有内容复制进去覆盖；
5. Summary 填：

```text
Upgrade to v2.8 all-in-one learning experience
```

6. Commit to main；
7. Push origin；
8. Vercel 自动重新部署。

## 注意

- 环境变量不用改；
- 数据库不用重新配置；
- 部署后刷新 Web App；
- 进入“学习”测试 Recall-first；
- 进入“薄弱”测试专项训练；
- 进入“章节”查看章节进度；
- 进入“管理”输入 ADMIN_PIN 查看管理概览。
